12자바스크립트 기초_함수의 개념과 활용

함수란? 일련의 내용을 삽입하여 사용자가 언제든 호출하여 실행할 수 있는 실행문이다.

  • 언제든 필요할 때 다시 실행이 가능
  • 코드의 중복을 피할 수 있고, 재사용이 가능하다.
  • 유지모수의 용이성이 높다.
  • 함수는 내장함수(propt, alert...미리만들어져있는것..)와 사용자 정의함수가 있다.
    1. 매개변수가 없는 일반함수
    function 함수명(){
      실행할 내용;
    }
    
    2. 매개변수가 있는 함수
    function 함수명(변수1, 변수2.....){
      실행할 내용;
    }

    3. 리턴값이 있는 함수
    function 함수명(변수1, 변수2.....){
      실행할 내용;
      return 실행결과;
    }
