05자바스크립트 연산자

자바스크립트 연산자는 사용자가 조건식을 만들어서 원하는 결과를 출력하고자 할 때 사용하는 기호이다.

  • 산술연산자 : +, -, *, /, %(나머지값)
  • 문자열 결합 연산다 : +(문자+문자, 문자+숫자 결합시 사용)
  • 비교연산자 : >, <, >=, <==, ==,===
  • 대입연산자 : =, +=, -+
  • 증감연산자 : ++(1씩 증가), ==(1씩 감소)
  • 논리연산자 : And(모두 조건에 만족시 실행), OR(여러 조건중에 하나만 만족시 실행)
  • 삼항조건연산자 : 조건식?조건에 맞는 결과:조건에 맞지 않는 결과
  • 타입연산자 : typeof(데이터의 형식을 알려주는 연산자)

실습예제 - 교보문고 책 이미지를 활용하여 좌, 우 버튼 클릭시 페이지 번호, 이미지 변경하기

네이버 쇼핑 카테고리, 교보문고 미리보기 책 콘텐츠, 여행사이트의 상품보기, 쇼핑몰의 상품목록 보기 등에서 활용할 수 있다.

구현방법 : 좌, 우측 방향버튼 클릭시 숫자 증가, 감소 기능을 구현하고, 변수값이 변경되면 이미지 파일명의 숫자도 같이 대입하여 변경하게 된다.
