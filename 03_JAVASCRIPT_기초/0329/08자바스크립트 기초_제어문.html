03자바스크립트 기초_제어문

제어문이란? 프로그램의 흐름을 제어하기 위한 실행문으로서 if, if else, else if문 3가지 종류가 있다.

  • if문 : 조건식에 만족하면 실행되는 조건문임.
  • if else문 : 조건식에 만족하면 true(합격), 만족하지 않으면 false(불합격)값이 출력되는 조건문임.
  • if else if문 : 조건식이 여러개일 경우 다양한 결과값을 얻기 위한 조건문이다.
