  • 네이버 오른쪽 콘텐츠 영역 - 실시간 뉴스, 날씨, 증시정보
  • 네이버 메인 배너광고, 오른쪽 작은 배너 (새로고침시)
  • 네이버 언론사 아이콘 목록 좌, 우 버튼 클릭시 콘텐츠 변경
  • 네이버 오른쪽 하단 '다크모드'버튼
  • 메뉴 카테고리 클릭시 콘텐츠 변경
  • 더보기, 펼치기 버튼 클릭시 콘텐츠 추가됨.
  • 글쓰기 버튼 클릭시 새창띄우기
  • 좌, 우 버튼
  • 알림버튼 클릭시 새콘텐츠 보이게 하기
  • '다른 추천보기' 버튼 클릭시 해당 콘텐츠 새로변경됨
  • 좌, 우 슬라이드
  • 새로고침 콘텐츠 자동변경됨

1. 자바스크립트 활용예제1 - 이미지 2장을 사용하여 버튼을 클릭하면 해당 이미지의 경로를 변경하여 전구사진 변경하기

  • onclick : 자바스크립트 이벤트의 종류중의 하나로서 사용자가 마우스로 버튼을 클릭을 하였을 경우 이벤트가 발생되어 스크립트가 동작한다.
  • onmouseover = onmouseenter : 사용자가 마우스를 어떤 대상에 올렸을 경우 스크립트가 실행
  • onmouseout = onmouseleave : 사용자가 마우스를 어떤 대상에 올렸다가 뺐을 경우 스크립트가 실행
  • getElementById(아이디명) : 사용자가 지정한 이름의 아이디를 선택, css의 #id선택자와 같음.
  • style : 자바스크립트에서 선택한 요소에 css서식을 적용하고자 할 때 사용
  • fontSize : 자바스크립트에서 css처럼 글자크기를 변경하고자 하는 스타일 속성.
