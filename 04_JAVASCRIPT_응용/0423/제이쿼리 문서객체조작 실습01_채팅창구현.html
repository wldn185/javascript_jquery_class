제이쿼리 문서객체조작 실습01_채팅창구현

채팅창 입력양식을 삽입하여 사용자가 글내용입력 후 '전송' 버튼 클릭시 글내용이 대화상자에 출력되게 하기

  • 채팅창 화면을 header, main, footer 영역으로 구성한다.
  • header영역은 브라우저 버튼, 사용자가 프로필영역, 소개글 작성
  • main영역은 상대방, 사용자가 입력한 글내용이 말풍선으로 출력되는 곳
  • footer영역은 사용자가 입력양식에 글을 입력하고 버튼 클릭시 변수에 담아 내용을 출력하도록 함.
  • 이때, 사용자가 글내용을 입력하지 않고 있으면 버튼 비활성화하기
  • 속성메서드: val, 클래스조작: addClass, 내용추가: append
