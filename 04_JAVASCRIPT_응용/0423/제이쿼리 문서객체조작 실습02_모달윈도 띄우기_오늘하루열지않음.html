제이쿼리 쿠키 플러그인

쿠키(cookie):서버에서 클라이언트에게 특정 정보를 저장하는 데이터

사용자 이름, 아이디, 방문 횟수 등을 기록하여 차후 접속시 활용

제이쿼리 쿠키 사용법
$.cookie('쿠키이름','쿠키값', {expires:만료일, path:'저장경로'});

제이쿼리 쿠키는 쿠키플러그인 파일을 다운로드하여 연결후 작성한다.
https://plugins.jquery.com/cookie/

  • expires:1 하루
  • var date = new Date(); //분단위로 설정할 때 date.setTime(date.getTime() + 1*60*1000); //1분 $.cookie('key', 'value', {expires: date, path : '/'});
