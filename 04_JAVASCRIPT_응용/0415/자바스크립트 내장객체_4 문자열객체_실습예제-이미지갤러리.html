자바스크립트 내장객체_4 문자열객체_실습예제-이미지갤러리

문자열 객체를 활용하여 탭메뉴 5개 클릭시 해당 문자열값을 변수에 담아 큰이미지가 변경될 수 있도록 src주소값을 변경한다.

  1. 독립적인 콘텐츠를 만들기 위한 article태그를 사용하여 박스를 만든다
  2. 썸네일 이미지 메뉴를 만들기 위한 ul, li태그를 작성한다.
  3. 큰사진이 보이도록 div태그를 작성한다
  4. script 문자열 객체를 통해 썸네일 이미지 클릭시 해당 src값을 변수에 담아 큰사진의 src속성값에 대입하여 이미지가 변경되도록 한다.
