18자바스크립트 내장객체_5 윈도우(window)_브라우저객체

브라우저 객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체이다.

Window객체

window객제는 브라우저 객체 중 '최상위 객체'로서 현재 실행되고 있는 브라우저 창을 나타낸다.

Window객체의 속성과 메서드

Window객체의 속성(property)종류
document
frameElement
innerHeight
innerWidth
location
name
screenX
screenY
scrollX, scrollY
sessionStorage : 사용자가 접속한 정보
Window객체의 메서드(method)종류
alert(); - 경고창 띄우기
blur(); - 현재창에서 포커스를 제거
close(); - 현재창을 당기
confirm(); - 확인/취소 버튼이 있는 확인창을 표시함.
focus(); - 현재창에 포커스(초점)를 부여함
moveBy(); - 현재창을 지정한 크기만큼 이동
moveTo(); - 현재창을 지정한 좌표로 이동
open(); - 지정한 문서를 열기
prompt(); - 사용자가 입력가능한 프롬프트창의 활성화함.
resizeTo(); - 동적으로 브라우저 창의 크기를 조절
scroll(); - 문서의 특정위치로 스크롤합니다.
sizeToContent(); - 내용에 맞게 창의 크기를 조절
stop(); - 로딩 중지

팝업창은 브라우저의 팝업차단 기능으로 인해 사용자가 보지 않을 가능성이 많아 요즘 팝업창을 띄우지 않고, 팝업 레이어, 모달윈도(css로 박스 포지션을 배치)를 활용하여 띄운다.

모달윈도(modal window)
모달윈도(modal windoe)는 사용자 인터페이스(UI)디자인 개념에서 자식 윈도에서 부모 윈도우로 돌아가기 전에 사용자의 상호동작을 요구하는 창을 말한다.
모달 윈도는 부모창을 배경으로 어둡게 하고, 자식요소가 화면의 중앙에 배치하도록 하며, 오른쪽 상단에 '닫기'버튼을 삽입하여 닫을 수 잇도록 한다.
