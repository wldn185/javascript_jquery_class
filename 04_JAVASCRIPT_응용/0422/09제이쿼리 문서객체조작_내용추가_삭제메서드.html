09제이쿼리 문서객체조작_내용추가_삭제메서드

DOM(문서객체모델)에서 문서안의 요소에 내용을 추가하거나 삭제할 수 있다.

클래스조작(addClass, removeClass, toggleClass, hasClass)

속성조작(attr)

내용조작(text html)

  • append(); - 사용자가 선택한 요소를 기준으로 안쪽으로 맨 뒤쪽에 내용을 추가
  • prepend(); - 사용자가 선택한 요소를 기준으로 안쪽으로 맨 앞쪽에 내용을 추가
  • after(); - 사용자가 선택한 요소를 기준으로 바깥쪽으로 맨뒤에 내용을 추가
  • before(); - 사용자가 선택한 요소를 기준으로 바깥쪽으로 맨앞에 내용을 추가

insertAfter(), insertBefore()를 사용하면 A라는 요소의 앞쪽에 B라는 요소를 배치할 수 있고, B라는 요소 뒤쪽에 A요소를 배치할 수 있다. after(), before()메소드와 기능은 같지만 사용법이 다르다.

예
$('A').after('B'); - A요소의 바깥쪽으로 뒤에 B내용을 추가한다.
$('B').insertAfter('A') - A요소의 바깥쪽으로 뒤에 B내용을 추가한다.

문서객체조작 - 내용을 삭제하기 우한 메서드(remove)

사용자가 선택한 요소의 내용만 삭제하고나, 아예 태그요소를 삭제할 수도 있다.

  • remove() - 사용자가 선택한 요소를 제거
  • empty() - 사용자가 선택한 요소의 내용만 제거

아래 remove버튼을 클릭하면 이 박스요소가 제거됩니다.
아래 empty버튼을 클릭하면 이 박스요소안에 있는 내용만 제거됩니다.
