노랑풍선, 네이버... 기타 메뉴 클릭시 해당영역안에서만 변경되는 탭메뉴 콘텐츠를 구현해본다.

노랑풍선 사이트의 '해외여행패키지 > 괌' 페이지의 탭메뉴 콘텐츠

탭메뉴 콘텐츠는 메뉴 클릭시 해당하는 콘텐츠가 바로 아래 나오면서 기존 콘텐츠는 숨겨지고, 영역안에서만 변경되는 콘텐츠이다.

사용하는 이유? 특정영역에서만 보여지기 때문에 좁은 공간에서 많은 콘텐츠를 보여주고자 할 때 사용.

구현방법
1. html 목록태글르 사용하여 목록(li)에 a요소 4개와 div요소를 사용하여 콘텐츠 영역 4개 만들고, css를 활용하여 첫번째 콘텐츠만 보이도록 스타일 서식을 지정한다.
2. tab메뉴(a요소)를 클릭하면 해당 콘텐츠(div)가 보이도록 하고, 나머지 콘텐츠들은 숨긴다. 이때 addClass()메소드를 사용한다.
3. 메뉴 서식은 선택한 메뉴에만 적요되도록 하고, 그외 나머지 메뉴는 적용된 서식을 제거한다. (removeClass)

사용할 메서드 - addClass(), removeClass(), siblings(), find(), css(), parent()
