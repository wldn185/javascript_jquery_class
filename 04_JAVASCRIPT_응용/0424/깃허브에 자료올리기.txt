깃허브에 자료 올리는 방법
1. GUI방식 (마우스로 드래그하여 올리는 방식)
 - 레포지토리(프로젝트 폴더) 생성
 - README파일 내용작성
 - 레포지토리에 파일 업로드
 - commit버튼 클릭하여 업로드 적용
 - 환경설정(settings) > page카테고리 브런치명 'main'선택, save
 - 화면 왼쪽 상단에 프로젝트명 클릭하여 메인화면으로 이동
 - 오른쪽 하단에 github-page 클릭하여 페이지 이동
 - url주소 생성된거 확인하고 클릭
   https://github.com/jeon9897/project01
 

2. CLI방식 (명령어를 입력하여 올리는 방식)
